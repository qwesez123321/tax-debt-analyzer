# Tax Debt Analyzer

Java-приложение для загрузки и анализа открытых данных ФНС РФ
о налоговой задолженности юридических лиц.

Источник данных:  
https://www.nalog.gov.ru/opendata/7707329152-debtam/

## Возможности
- автоматическое получение ссылки на ZIP-архив с сайта ФНС
- загрузка и обработка ZIP-архива
- потоковый парсинг XML (StAX)
- агрегация данных по компаниям и типам налогов
- расчёт статистики:
    - количество компаний
    - максимальная сумма долга одной компании
    - максимальный долг по каждому типу налога
    - средняя сумма долга

## Требования
- Java 17+
- Maven 3.8+
